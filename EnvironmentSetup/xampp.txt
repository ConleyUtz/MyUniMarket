We'll be using XAMPP for local environment setup.
XAMPP is an open source application with support for Windows, OSX and Linux. 
XAMPP provides running MySQL server and support for PHPMyAdmin.

Steps to setup your local environment:

1) Download XAMPP from https://www.apachefriends.org/download.html for your respective OS.

2) Start ApacheWebServer and MySQL server.

3) Now, localhost or 127.0.0.1 will respond to changes in htdocs folder inside xampp.
